1. HTTP request methods: GET, HEAD, POST, PUT, DELETE, OPTIONS, TRACE

2. HTTP status codes: 404 (Not found), 200 (OK), 202 (Accepted), 301 (Moved permanently), 400 (Bad request), 401 (Unauthorized), 403 (Forbidden), 418 (I'm a teapot)

3. Startup class used for: configure services (Razor Pages, DBContext, ...) and configure middleware pipeline (routing, endpoints)

4. HSTS (HTTP Strict Transport Security): web browser connect to a server only via HTTPS (TLS/SSL)

5. Static HTML pages: app.UseDefaultFiles(); app.UseStaticFiles();

6. Dynamic page: .cshtml (Razor) @page @function{ }

7. Shared layouts: Pages\Shared\_Layout.cshtml - @RenderBody

8. Markup: .cshtml (@model MyModel ... @Model.Suppliers); Code-behind: .cshtml.cs (class MyModel : PageModel)

9. EFCore in ASP.NET Core: services.AddDBContext(); OnGet - get data; OnPost - set data

10. Partial: .cshtml - partial name="_Employee" model="employee" Pages\_Employee.cshtml @Model.FirstName
